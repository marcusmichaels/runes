Modulo

/ˈmɒdjʊləʊ/

%

a % b gives you the remainder after dividing a by b. Three ways to see what that means.

Divide, keep the leftovers

Imagine a dots you want to pack into groups of b. Full groups (teal) tile away; what's left over (rust) is the modulo.

The cycle

Keep counting up n. The value of n % b wraps back to 0 every time n reaches b. It's how clocks work — after 11 comes 0, not 12.

Cycling through an array

The classic practical use: arr[i % arr.length] lets a counter safely index an array forever, wrapping back to the start each time it reaches the end.
