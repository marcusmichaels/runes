Addition

/əˈdɪʃ(ə)n/

+

a + b gives you their sum — simple with numbers, less simple when strings show up.

On a number line

Two moves along the line. First a, starting from zero. Then b, starting where a landed. Where you end up is a + b.

When strings show up

If either operand is a string, + stops being arithmetic and becomes concatenation. JavaScript coerces the other side to a string and glues them together. Click the examples.
